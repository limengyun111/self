# My test project 
<img src="https://img.shields.io/badge/node-v20.18.1-blue" />  <img src="https://img.shields.io/badge/npm-v10.8.2-blue" />

This is a simple project with three routes that can be started using express or webpack.


## Start up
```
cd ./self/back-end
npm run dev
or
cd ./self/front-end
npm run dev
```

## Build
```
cd ./self/front-end
npm run build:copy
```
